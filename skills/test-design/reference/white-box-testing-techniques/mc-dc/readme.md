Teste de Condição/Decisão Modificada (MC/DC): Trata predicados lógicos que contêm múltiplas condições atômicas em uma só decisão, garantindo que a alteração de apenas uma condição mude isoladamente o resultado da decisão. Para que serve: Oferece excelente cobertura de código combinacional com número suportável de testes, amplamente empregado em indústrias críticas à segurança e à vida humana (aeroespacial e automotiva).

usa a técnica de teste de desicao e instrucao, porém dos fluxos que fazem sentido para ser mais rapido a criação de testes. essa skill aqui provavelmente vai consumir das de decisao e instrucao.
